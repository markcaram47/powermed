# PowerMed Frontend

React frontend application for PowerMed.

## Quick Start

```bash
# Install dependencies (if not already installed)
npm install

# Start development server
npm start
```

The app will open at `http://localhost:3000`

## Environment Variables (Optional)

Create a `.env` file in this directory:

```
REACT_APP_API_URL=http://localhost:5001/api
```

If not set, it defaults to `http://localhost:5001/api`

## Available Scripts

- `npm start` - Start development server
- `npm run build` - Build for production
- `npm test` - Run tests

## Notes

- Make sure the backend server is running on port 5001 before using the admin panel
- Click the PowerMed logo in the header to access admin login
